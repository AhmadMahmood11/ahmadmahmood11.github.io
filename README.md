# Ahmad Mehmood Portfolio Website

This is a GitHub Pages ready portfolio website.

## Files

- `index.html` - Main website file with HTML, CSS, and JavaScript included.
- `.nojekyll` - Keeps GitHub Pages from processing the site with Jekyll.

## How to Upload to GitHub

1. Create a new GitHub repository.
2. Upload `index.html`, `README.md`, and `.nojekyll` to the repository.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save and wait for GitHub Pages to publish the website.
