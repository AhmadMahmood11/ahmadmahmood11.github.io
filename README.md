# Ahmad Mahmood — Portfolio

Dark, editorial portfolio inspired by PixelPier NYC (VamTam theme).

## Files

```
index.html   — All sections (Hero, About, Career, Expertise, Work, Awards, Reviews, Contact)
style.css    — Full styles with dark theme, animations, responsive layout
script.js    — Custom cursor, scroll reveal, review slider, mobile menu
```

## How to upload to your GitHub repo

1. Download all 3 files (`index.html`, `style.css`, `script.js`)
2. Go to your repo: **https://github.com/ahmadmahmood11/ahmadmahmood11.github.io**
3. Delete (or replace) the existing files
4. Upload these 3 files to the **root** of the repo
5. Commit with message: `Update portfolio design`
6. Your site will be live at: **https://ahmadmahmood11.github.io/**

> GitHub Pages auto-deploys from the main branch. Changes go live in ~1 minute.
