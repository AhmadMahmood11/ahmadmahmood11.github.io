## How to upload on GitHub Pages

1. Open your GitHub portfolio repository.
2. Delete old files or keep a backup first.
3. Upload these 3 files into the root folder of the repository.
4. Commit changes.
5. Go to `Settings > Pages`.
6. Set source to `Deploy from a branch`.
7. Select branch `main` and folder `/root`.
8. Save.
9. Your website will update in a few minutes.
